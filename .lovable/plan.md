
# План: сайт «Бетонное полотно» — ООО ЗЭМ «Электровибромашина»

Современный многостраничный B2B-сайт с фото-плейсхолдерами и плавными анимациями при скролле. Без бэкенда на этом этапе — форма заявки покажет успешное сообщение (подключим Lovable Cloud + email позже).

## Структура сайта (маршруты)

```
/                  Главная — hero, УТП, краткое о продукте, сферы применения, CTA
/product           О продукте — суть материала, принцип работы, преимущества, характеристики
/applications      Сферы применения — 6 карточек (склоны, каналы, растительность, и т.д.)
/delivery          Формы поставки — компактные / большие / широкие рулоны
/installation      Технология монтажа — 6 шагов
/about             О компании — описание, реквизиты, директор
/contacts          Контакты — телефоны, email, адрес, карта Яндекс, форма заявки
```

В шапке: логотип + навигация по 7 разделам + телефон-кнопка.
В подвале: контакты, реквизиты, время работы, копирайт.

## Дизайн

- **Стиль**: современный промышленный B2B с акцентом на инженерную надёжность. Крупные фото объектов, светлый фон, чёткая типографика, плавные fade-in / scale-in анимации при скролле.
- **Палитра**: белый фон + цвета логотипа (заберём с muftatormoz.ru). До получения логотипа использую нейтральные индустриальные тона (тёмно-серый/графит + оранжево-стальной акцент) и затем перекрашу под реальный логотип.
- **Типографика**: пара «Manrope (заголовки) + Inter (текст)» — строгая, инженерная, хорошо читается на русском.
- **Компоненты**: hero с фоновым изображением, секции преимуществ с иконками-цифрами (10×, 24ч, 120 лет, 300 циклов), карточки применений с hover-эффектом, степпер для монтажа, таблица характеристик рулонов.

## Технические детали

- TanStack Start, файловый роутинг в `src/routes/`
- Tailwind v4 токены в `src/styles.css` (цвета логотипа как semantic tokens)
- Шрифты через `@fontsource/manrope` и `@fontsource/inter`
- Общий Layout-компонент (Header/Footer) подключается в `__root.tsx`
- SEO: уникальные `head()` (title, description, og:*) на каждой странице, lang="ru"
- Анимации: `animate-fade-in`, `hover-scale`, IntersectionObserver для секций при скролле
